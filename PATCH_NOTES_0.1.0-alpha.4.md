# The Birth of Steve 0.1.0-alpha.4

This update focuses on room readability, interaction, item presentation, and
asset provenance.

## Highlights

- A new dedicated creative tab contains every block and item from the mod,
  including each Memory Plate scene.
- Archive rooms now use full-block weathered floor patches. The partial stair
  shapes that exposed dark gaps and invisible-looking floor sections are gone.
- Vertical room connections now start with one additional lower stair and use
  four-block-deep side walls, closing the remaining gaps around stair runs.
- Players can break interior props and set dressing during active dungeon runs.
  The generated room walls and floor remain protected.
- Crate-break audio starts immediately instead of carrying a noticeable silent
  lead-in.
- Graveyard props now render sharply in hand through item-atlas models while
  retaining their detailed placed-block models.
- Standalone item textures were cleaned into crisp 32-pixel sprites with hard
  alpha edges and blur disabled.

## Content

- Added 38 directional graveyard props: gravestones, grave crosses, coffins,
  skeletal remains, cemetery fencing, and flowers.
- Added matching block items, blockstates, loot sidecars, translations, and
  creative-tab entries.

## Asset and release audit

- Removed generated CurseForge banner/icon artwork and the packaged logo
  reference.
- Completed the release-authorization audit for bundled prop assets.
- Updated the mod version to `0.1.0-alpha.4`.

## Compatibility

- Minecraft Java Edition 26.1.2
- NeoForge 26.1.2.83 or newer for Minecraft 26.1.2
- Java 25
